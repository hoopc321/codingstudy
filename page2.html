변수 선언

변수란 데이터를 저장하기 위해 이름을 할당받은 메모리 공간을 의미한다.
사용할 데이터에 따라 정수형, 실수형, 문자형 변수의 타입을 고려하여 사용해야하며 변수의 데이터는 바뀔 수 있다.

예제

#include<stdio.h>

int main(){ 

int a;  double b;  char c;
int a2=10;  double b2=3.141592;  char c2='c';

return 0;
}




변수의 이름은 영어와 언더바, 숫자를 사용하여 선언 가능하며 C언어 내에서 정의된 몇몇 키워드로는 사용 불가능하다.(ex. case char goto 등)
변수를 선언할 때, 값을 지정하며 선언할 수 있다.(예제)
또, 값의 메모리 크기에 따라 적절한 선언문을 지정해야한다.
