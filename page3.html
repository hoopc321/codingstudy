입출력

scanf() 함수와 printf() 함수는 C언어 입출력 함수 중 가장 많이 사용되는 함수이다.
아래 예제는 a와 b라는 변수를 만들어 a에는 사용자가 입력하는 값을 지정해 출력하고 b는 10을 지정해 출력하는 코드다.

예제

#include<stdio.h>

int main(){

int a, b;

scanf(%d, &a);
b=10;

printf("%d", a);
printf("%d", b);

return 0;
}
사용자가 5를 입력할 경우 출력값 :
5
10

printf() 함수에서 정수를 출력할 목적이기에 %d를 사용하였고 "" 뒤에 {, 변수}로 출력할 변수를 지정한다.
{, 변수} 형태에서 변수를 a+b 등의 수식으로 넣을 수 있다.

scanf() 함수도 prinf() 함수와 형태가 거의 동일하지만 변수 하나를 지정해 사용하여야한다.
또, 입력 받을 변수 앞에 &(주소연산자)를 붙이지 않으면 오류가 난다.




서식문자




예제의 %d는 10진수 정수를 입출력하기 위한 서식문자로, 원하는 입출력 데이터 형태에 따라 바꾸어 사용할 수 있다. 모든 변수를 사용할 때에는 사용하려는 데이터의 크기와 형태를 고려하여 서식 문자를 설정해야 한다.
