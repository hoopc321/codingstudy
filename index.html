C 프로그램의 구조

C 프로그램의 가장 기본적인 구조이다.
항상 이런 구조를 사용하지는 않지만 대부분의 코드가 예제의 코드 틀을 사용한다.

예제

#include<stdio.h>

int main(){


     return 0;
}


#include<stdio.h>를 통해 stdio.h라는 헤더파일을 불러와 여러 함수를 사용할 수 있도록 하는 코드다.
C 언어는 int main()을 사용하여 main이라는 정수형 함수를 선언, 함수의 리턴(결과)값을 운영체제가 가장 먼저 받아들이게 한다.
return 0는 0의 값을 반환한다는 뜻이고 메인함수에서는 코드를 끝내는 역할로 사용된다.
또, 모든 구문이 끝날 때 ;(세미콜론)을 사용하여 코드들을 구분하며 영어의 대소문자를 구분한다.
\n을 이용하여 줄구분이 가능하다.
